## Math
This repository contains some common math formulas that can be used to reduce time complexity from O(n) to O(1).

### Sum of n Terms
    S(n) = n * (n + 1) / 2

### Sum of n Odd Terms
    S(n) = n * (n + 1) / 2
    S(2n-1) = n * (2n - 1 + 1) / 2
    S(2n-1) = n * 2n / 2 = n * n
